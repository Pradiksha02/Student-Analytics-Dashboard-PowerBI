# 🎓 Student Analytics Dashboard

## 📌 Overview

This project is an interactive Power BI dashboard developed to analyze student demographics, attendance, faculty, and course information. It provides meaningful insights through interactive visualizations and KPIs.

---

## 📊 Dashboard Features

- Total Students
- Total Faculty
- Total Courses
- Average Attendance
- Department-wise Student Analysis
- Gender Distribution
- Academic Year Analysis
- Attendance Analysis

---

## 🛠 Tools & Technologies

- Power BI
- Power Query
- DAX
- Microsoft Excel
- Data Visualization
