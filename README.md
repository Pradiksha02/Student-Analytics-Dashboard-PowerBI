# 🎓 Student Analytics Dashboard using Power BI

## 📌 Project Overview

The **Student Analytics Dashboard** is an interactive Power BI project designed to analyze student demographics and attendance data. The dashboard provides meaningful insights through KPI cards, charts, and interactive visualizations, helping users understand student distribution across departments, academic years, and gender.

---

## 🎯 Objectives

* Analyze student demographic information.
* Visualize attendance trends.
* Compare department-wise student distribution.
* Monitor key academic metrics using KPIs.

---

## 🛠 Technologies Used

* Power BI
* Power Query
* DAX
* Microsoft Excel

---

## 📊 Dashboard Features

* 👨‍🎓 Total Students
* 👨‍🏫 Total Faculty
* 📚 Total Courses
* 📈 Average Attendance
* Student Distribution by Department
* Student Distribution by Gender
* Students by Academic Year
* Attendance Analysis
